# Mae MacDonald – Academic Website

This is a personal academic website hosted on GitHub Pages.

## Setup

1. Push this repository to GitHub as `yourusername.github.io` (or any repo name with GitHub Pages enabled).
2. Replace `img/mae_macdonald.jpg` with your headshot photo.
3. Replace `cv/macdonald_cv.pdf` with your actual CV file.
4. Update the Google Scholar link in `index.html` with your real Google Scholar profile URL.
5. Enable GitHub Pages in your repository settings (Settings → Pages → Source: main branch).

## Structure

```
├── index.html              # Main homepage
├── stylesheets/
│   └── styles.css          # Site styling (Lato font, fixed sidebar layout)
├── img/
│   └── mae_macdonald.jpg   # Your headshot (add this)
├── cv/
│   └── macdonald_cv.pdf    # Your CV (add this)
└── README.md               # This file
```

## Customization

- Edit `index.html` to update your bio, publications, and links.
- The layout uses a fixed left sidebar (270px) with your photo and contact info, and a scrollable right content area (500px).
- Links are styled in blue (#3498db). Adjust in `stylesheets/styles.css`.
